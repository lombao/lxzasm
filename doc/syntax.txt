How to express Hexadecimal values
====================================
You have all these valid possiblities, all of this represent the same value.
Chose the format you prefer

   LD HL, &4000 
   LD HL, #4000
   LD HL, 0x4000
   LD HL, 4000H

How to express Binary values
=============================
Let's upload 16 in binary format into the register A:

   LD A,00010000b
